# About

I learn assembler by the ChatGPT o1 and the ChatGPT 4o. I use Ubuntu, NASM, and Docker to run the code.

# Manage container

## Build container
```bash
docker compose build
```

## Run container
```bash
docker compose up -d
```

## Login to container
```bash
docker compose exec app bash
```

## Stop container
```bash
docker compose down
```

## Remove container
```bash
docker compose down --volumes
```
