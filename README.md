# Manage container

## Build container
```bash
docker compose build
```

## Run container
```bash
docker compose up -d
```

## Login to container
```bash
docker compose exec app bash
```

## Stop container
```bash
docker compose down
```

## Remove container
```bash
docker compose down --volumes
```
